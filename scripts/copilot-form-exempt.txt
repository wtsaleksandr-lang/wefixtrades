# here only when it has no user-fillable form — a dashboard, a read-only
# list/detail view, a wizard driven by button choices — or when filling it
# with AI is deliberately disallowed (password / credential forms).

# If you add a new page WITH an editable form, do NOT list it here — wire it
# with useCopilotForm() instead. If a listed page later gains a fillable
# form, remove it from this list and wire it (the guard will remind you).
