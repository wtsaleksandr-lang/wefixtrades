# implement-design

Using the approved design direction, implement the new design into my current project.

Workflow:
1. analyze the existing codebase
2. identify affected files
3. make minimal safe edits
4. preserve current functionality unless I explicitly ask to change it
5. reuse existing components where possible
6. add styling, layout, animation, and interaction updates as needed
7. run relevant checks after changes
8. fix any issues introduced

Prioritize:
- clean modern UI
- responsive layout
- reusable components
- production-safe code
