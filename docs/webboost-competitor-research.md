# WebBoost™ — Competitor Research Dataset

## 1. Competitor Landscape (12 Companies)

Competitors fall into three categories:
- **Category A**: One-time / project-based website optimization services
- **Category B**: Ongoing WordPress maintenance + speed optimization plans
- **Category C**: Self-serve speed optimization tools (SaaS)

## 2. Pricing Patterns

### By Service Type

| Service Type | Price Range | Model |
|-------------|-----------|-------|
| **Fiverr / budget one-time fix** | $30–$250 | One-time |
| **Small agency one-time optimization** | $140–$500 | One-time project |
| **Mid-agency one-time project** | $500–$2,000 | One-time project |
| **Enterprise one-time project** | $2,000–$12,000 | One-time project |
| **Monthly WP maintenance + speed** | $59–$197/mo | Monthly subscription |
| **Premium ongoing optimization** | $399–$750+/mo | Monthly subscription |
| **SaaS auto-optimization tool** | $8–$100/mo | Monthly SaaS |
| **Full SEO retainer (includes speed)** | $500–$3,500/mo | Monthly retainer |

### Low-End vs Mid vs Premium

| Tier | Price | What You Get | Who Buys |
|------|-------|-------------|----------|
| **Low-end** | $30–$195 one-time | Basic caching, image compression, plugin setup. Score improvement only. No audit, no SEO, no report | DIYers, micro businesses |
| **Mid-market** | $200–$500 one-time OR $59–$149/mo | Full speed optimization, CDN, image optimization, Core Web Vitals fixes, basic report, some ongoing maintenance | Small businesses, trades, local businesses |
| **Premium** | $500–$2,000+ one-time OR $197–$750+/mo | Deep technical audit, code-level fixes, server optimization, ongoing monitoring, dedicated support, SEO included | Growing businesses, e-commerce, multi-location |
| **Enterprise/Agency** | $2,000–$12,000+ project OR $3,000+/mo retainer | Full technical SEO audit + fixes, content strategy, speed + SEO + CRO, custom reporting | Large businesses, franchises |

## 4. How Agencies Package This Service

| Packaging Pattern | Examples | Notes |
|------------------|----------|-------|
| **Speed-only one-time fix** | WPSpeedFix, Hues Agency, Fiverr | Cheapest. Fix and forget. No ongoing relationship |
| **Speed + maintenance bundle** | WP Buffs, FixRunner | Monthly fee covers speed + updates + security + backups. Sticky recurring revenue |
| **Speed as part of SEO package** | WebFX, SEO Web Marketing | Speed optimization is a line item inside a larger $500–$3,000/mo SEO retainer |
| **Automated SaaS tool** | NitroPack | Self-serve. Install plugin, pay monthly. No human involvement |
| **Audit → fix → retain** | Optimocha, Seahawk Media | Free or paid audit → one-time fix project → ongoing retainer upsell |
