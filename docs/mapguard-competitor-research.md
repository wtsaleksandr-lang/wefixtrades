## 5. Proposed MapGuard™ Setup Pricing & Packaging

### A. Suggested Price: **$397 one-time**

**Rationale:**
- Above the Fiverr/budget tier ($25–$100) — signals quality
- At the upper-mid range, competing with 1Solutions ($250) and GMB Gorilla ($499)
- Below the agency threshold ($500+) — accessible for trades businesses
- High perceived value with professional delivery
- Strong margin at outsourced or internal delivery costs

### B. What MUST Be Included to Justify $397

| Deliverable | Description |
|-------------|-------------|
| **GBP Audit** | Full audit of existing profile (or competitor audit if new) — checklist-based scoring |
| **Profile Claiming & Verification** | Handle full claiming/verification process if needed |
| **Category Optimization** | Primary + secondary category selection based on keyword research |
| **NAP Consistency Check** | Verify Name, Address, Phone matches across Google, Facebook, Yelp (top 5 directories) |
| **Business Description Copywriting** | SEO-optimized, 750-char description with target keywords |
| **Service/Product Listings** | Add all services with descriptions to GBP |
| **Photo Optimization** | Upload & optimize 10–15 photos (geo-tagged, properly named) — client provides or stock |
| **Q&A Seeding** | Pre-populate 5 Q&As with keyword-rich answers |
| **Google Posts** | Create 2 initial Google Posts (offer + update) |
| **Completion Report** | Before/after audit scorecard, what was done, recommendations for ongoing |

### C. What Can Be Outsourced vs Kept Internal

| Task | Outsource? | Keep Internal? | Notes |
|------|-----------|---------------|-------|
| GBP Audit | Yes (checklist-based) | Preferred | Quick with a template, 15–20 min |
| Claiming & Verification | No | Yes | Requires client trust & credentials |
| Category Research | Yes | Either | Keyword tool + competitor check |
| NAP Check | Yes | Either | Can be done via BrightLocal or manual |
| Description Writing | Yes | Either | Freelancer can write from template |
| Service Listings | No | Yes | Needs client-specific detail |
| Photo Upload & Optimization | Yes | Either | Geo-tagging is simple, bulk upload |
| Q&A Seeding | Yes | Either | Template-based |
| Google Posts | Yes | Either | Template + Canva graphic |
| Completion Report | No | Yes | Brand-building touchpoint, keep in-house |
